Número

O número é um objeto da matemática usado para descrever quantidade, ordem ou medida.

Representação

O sistema mais utilizado para representar um número é o sistema decimal, em que a base é 10 e utiliza-se os algarismos de 0 a 9. O sistema decimal é classificado como notação posicional pois é um modo de representação numérica na qual o valor de cada algarismo depende da sua posição relativa na composição do número.

Binário

Outro sistema bastante utilizado é o binário ou de base 2 (1, 2, 4, 8, 16, ...) que tem como algarismos os números 0 e 1 e que também é uma notação posicional.

Bit

Cada 0 ou 1 é chamado de bit (abreviatura de Binary Digit). Neste exemplo o número 45 é representado utilizando 6 bits. Devido a essa característica, o sistema binário é amplamente utilizado em computadores, pois permite fazer operações lógicas e aritméticas usando-se apenas dois dígitos ou dois estados (sim ou não, verdadeiro ou falso, tudo ou nada, ligado ou desligado, 1 ou 0). Toda a electrónica digital e computação estão baseadas nesse sistema binário sendo possível representar por circuitos electrónicos digitais (portas lógicas) os números, caracteres, realizar operações lógicas e aritméticas. Os programas de computadores são codificados sob forma binária e armazenados nas memórias, discos, etc, sob esse formato.

Jogo Mensagem Secreta

Todos nós estamos habituados a utilizar computadores e telemóveis. Mas será que sabemos como é que estas máquinas comunicam? Pois é, estas máquinas transferem informação (como texto, música, imagens) utilizando apenas dois dígitos: o 0 e o 1. A cada letra do alfabeto podemos associar um número. Se escrevermos esse número em numeração binária, podemos escrever uma mensagem secreta. Neste jogo, vais ter que descobrir a mensagem secreta que está escrita na vertical. Cada letra pode ser identificada descobrindo o número que está representado no sistema binário em cada linha. Clica aqui para jogar!
